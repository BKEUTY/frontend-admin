## English - Administration System

### 🏢 Overview
BKEUTY Admin is a sophisticated **Enterprise Resource Planning (ERP)** portal designed to centralize and automate business operations. It empowers administrators with deep insights and granular control over the entire commerce lifecycle.

### 🛠 Core Technical Stack
- **Admin Web:** React 19, Vite, Ant Design 6, Chart.js, Recharts.
- **Admin Mobile:** React Native, Expo 50+, React Navigation 7.
- **Security:** Role-Based Access Control (RBAC) via Keycloak.
- **Data Integration:** RESTful APIs with TanStack Query for optimistic UI updates.

### 📊 Strategic Modules
- **Analytics Dashboard:** Dynamic charts for revenue, orders, and top-selling products.
- **Reports & Invoice:** Complex reporting system with time-based filtering and Excel (XLSX) export. Includes feature to generate PDF payment invoices for customers.
- **Promotion Management:** Flexible interface for creating and managing product promotions and vouchers.
- **User Management:** Centralized view and management of customer profiles synchronized with Keycloak.

| :--- | :--- | :--- | :--- |

| **Admin Mobile** | Expo Metro | 8082 | `npx expo start` |

## Tiếng Việt - Hệ thống Quản trị

### 🏢 Tổng quan
BKEUTY Admin là cổng thông tin **Quản trị doanh nghiệp (ERP)** tinh vi được thiết kế để tập trung hóa và tự động hóa các hoạt động kinh doanh. Hệ thống cho phép người quản trị nắm bắt thông tin chuyên sâu và kiểm soát chi tiết toàn bộ vòng đời thương mại.

### 🛠 Công nghệ Cốt lõi
- **Admin Web:** React 19, Vite, Ant Design 6, Chart.js, Recharts.
- **Admin Mobile:** React Native, Expo 50+, React Navigation 7.
- **Bảo mật:** Quản lý truy cập dựa trên vai trò (RBAC) thông qua Keycloak.
- **Tích hợp dữ liệu:** RESTful APIs với TanStack Query giúp cập nhật giao diện tối ưu (Optimistic UI).

### 📊 Module Chiến lược
- **Dashboard Phân tích:** Biểu đồ động hiển thị doanh thu, đơn hàng và các sản phẩm bán chạy nhất.
- **Báo cáo & Hóa đơn:** Hệ thống báo cáo phức tạp cho phép lọc theo thời gian và xuất file Excel (XLSX). Hỗ trợ tạo hóa đơn thanh toán PDF cho khách hàng.
- **Quản lý Khuyến mãi:** Giao diện linh hoạt để tạo và quản lý các chương trình khuyến mãi sản phẩm và voucher.
- **Quản lý Người dùng:** Xem và quản lý tập trung danh sách khách hàng được đồng bộ hóa trực tiếp từ Keycloak.
