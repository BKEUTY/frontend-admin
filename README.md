<a name="english"></a>
## English

### Overview
BKEUTY Admin is the centralized management portal of the BKEUTY platform. It is designed exclusively for administrative staff to manage products, orders, appointments, and business analytics with a secure, professional interface.
1. **Web Admin Dashboard:** A powerful management interface built with React 19.
2. **Mobile Admin App:** On-the-go management experience built with React Native and Expo.

### Key Features (Admin Focus)

#### 1. Secure Administrative Access
- **Encapsulated Login:** Dedicated authentication for administrators only.
- **Strict Role Enforcement:** Exclusively handles logic for the `ADMIN` role. Unauthorized access is strictly denied.
- **Admin Recovery:** Secure password reset flow designed for administrative accounts.

#### 2. Professional Management UI
- **Professional Aesthetic:** Clean, data-focused design with an "Admin Navy" identity (#2c3e50).
- **Responsive Management:** Fully optimized Web dashboard for desktop and mobile devices.
- **Efficient Workflows:** Streamlined interfaces for managing large catalogs and order volumes.

#### 3. Core Administrative Functionality
- **Catalog Management:** Create, edit, and update products, categories, and inventory.
- **Order & Appointment Control:** Real-time management of customer orders and service bookings.
- **Business Analytics:** Overview of revenue, staff performance, and trending products.
- **Internal Tools:** Staff management and system-wide reporting.

### Technical Stack
- **Web:** React 19, Ant Design 6, Context API, React Router v7.
- **Mobile:** React Native, Expo 50+, React Navigation v7, Async Storage.

## Tiếng Việt

### Tổng quan
BKEUTY Admin là cổng quản trị tập trung của nền tảng BKEUTY. Đây là công cụ chuyên biệt dành cho đội ngũ quản lý để điều hành sản phẩm, đơn hàng, lịch hẹn và theo dõi báo cáo kinh doanh qua giao diện bảo mật, chuyên nghiệp.
1. **Website Quản trị:** Giao diện điều hành mạnh mẽ trên React 19.
2. **Ứng dụng Mobile Admin:** Công cụ quản lý linh hoạt trên React Native và Expo.

### Tính năng chính (Dành cho Quản trị)

#### 1. Truy cập Hành chính Bảo mật
- **Đăng nhập Độc lập:** Hệ thống xác thực dành riêng cho quản trị viên.
- **Kiểm soát Quyền hạn Tuyệt đối:** Xử lý logic duy nhất cho role `ADMIN`. Truy cập trái phép được ngăn chặn triệt để.
- **Khôi phục Tài khoản Admin:** Quy trình cấp lại mật khẩu bảo mật cho tài khoản cấp quản lý.

#### 2. Giao diện Quản lý Chuyên nghiệp
- **Thẩm mỹ Chuyên sâu:** Thiết kế tập trung vào dữ liệu với tông màu "Admin Navy" (#2c3e50).
- **Quản lý Linh hoạt:** Dashboard Web tối ưu trên cả máy tính và thiết bị di động.
- **Quy trình Tối ưu:** Giao diện được thiết kế để xử lý nhanh khối lượng lớn sản phẩm và đơn hàng.

#### 3. Chức năng Cốt lõi
- **Quản lý Danh mục:** Tạo, sửa và cập nhật sản phẩm, ngành hàng, tồn kho.
- **Điều hành Đơn hàng & Lịch hẹn:** Quản lý thời gian thực các đơn hàng và lịch đặt dịch vụ từ khách.
- **Báo cáo Kinh doanh:** Tổng quan về doanh thu, hiệu suất nhân viên và sản phẩm bán chạy.
- **Công cụ Nội bộ:** Quản lý nhân sự và xuất báo cáo hệ thống.

### Nền tảng Kỹ thuật
- **Web:** React 19, Ant Design 6, Context API, React Router v7.
- **Mobile:** React Native, Expo 50+, React Navigation v7, Async Storage.
